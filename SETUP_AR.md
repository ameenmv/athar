# 🔮 تثبيت واستخدام "أثر" (Athar)

"أثر" هي أداة ذكاء اصطناعي بتشتغل كـ (MCP Server) جوة Antigravity IDE. وظيفتها إنها تتابع أخطائك، وتفهم أنت بتعطل في إيه، وتحفظلك دروس بناءً على كودك الحقيقي عشان متغلطش نفس الغلطة تاني. 

كل ده بيحصل أوتوماتيك وإنت بتكود!

---

## 🚀 التثبيت في خطوتين بس

عشان الأداة تشتغل معاك، محتاج بس يكون عندك **Node.js 20+**.

### 1️⃣ ثبّت الأداة
افتح التيرمنال واكتب الأمر ده عشان تثبتها على جهازك كله:
```bash
npm install -g athar-mcp
```

### 2️⃣ اربطها بـ Antigravity IDE
اكتب الأمر ده عشان الأداة تعرّف نفسها لـ Antigravity:
```bash
athar setup
```
> 💡 **ملاحظة:** بعد الخطوة دي، افتح Antigravity واعمل **Refresh** للسيرفرات (أو اقفله وافتحه تاني). هتلاقي `athar` ظهرت في قايمة الـ MCP Tools.

خلاص كده! الأداة جاهزة وشغالة في الخلفية بتابع وتتعلم معاك. 🎉

---

## 🎮 إزاي بتستخدمها؟ 

ببساطة.. أنت مش محتاج تعمل حاجة! 
طول ما أنت بتكود وبتسأل المساعد الذكي عن مشاكلك، المساعد لوحده هيعرف إمتى الدرس يستاهل يتحفظ، وهيحفظه.

لكن عشان تتابع مستواك وتشوف الدروس دي، عندك 4 أوامر بسيطة في التيرمنال:

### 1. الداشبورد السحرية ✨
عشان تتفرج على إنجازاتك وكل الدروس اللي اتعلمتها في واجهة Dark Mode شيك جداً، اكتب:
```bash
athar dashboard
```
هيفتحلك الموقع على `http://localhost:3333`، وهتلاقي فيه:
- إحصائيات مستواك.
- الدروس بتاعتك.
- مقارنة شكل الكود بتاعك قبل وبعد (Bad vs Good code).

### 2. شوف حالتك بسرعة 📊
لو مش عاوز تفتح الداشبورد وعاوز نظرة سريعة من التيرمنال:
```bash
athar status
```

### 3. وقت المراجعة 🧠
عشان المعلومة تثبت (باستخدام التكرار المتباعد Spaced Repetition):
```bash
athar review
```
الأداة هتسألك أسئلة عن الدروس المستحقة للمراجعة، وتخليك تقيّم إجابتك، وبناءً عليه تحدد هتسألك تاني إمتى.

### 4. قلّب في دفاترك القديمة 📚
لو عاوز تدور على درس معين، أو تفلتر بلغة معينة:
```bash
athar list
athar list --language typescript
athar list --tag react
athar list --search "useEffect"
```

## ❓ أسئلة شائعة في السريع

- **حاجتي بتترفع فين؟** 
  ولا أي حتة! كل بياناتك ودروسك بتتحفظ في داتا بيز `SQLite` صغيرة على جهازك أنت وبس.
  
- **بفلوس؟**
  لا، الأداة مجانية بالكامل (Open Source).

- **لو عاوز أمسح كل حاجة وأبدأ على نضافة؟**
  احذف الملف ده: `~/.local/share/athar/lessons.db` (في لينكس)، أو ما يعادله في الويندوز والماك.
